# SQL Sistema de Vendas

Projeto desenvolvido para praticar SQL e MySQL através da criação de um banco de dados de vendas.

## Funcionalidades
- Criação de banco de dados
- Criação de tabelas
- Inserção de dados
- Consultas SQL
- Filtros com WHERE
- Busca com LIKE
- Intervalos com BETWEEN
- Agregações com COUNT, SUM e AVG
- Relatórios com GROUP BY e ORDER BY

## Tecnologias
- MySQL
- SQL
